# Olympics Data Analysis Dashboard | Power BI

## Project Overview

This project is an interactive Olympics Data Analysis Dashboard built using Power BI to analyze historical Olympic data and uncover meaningful trends in athlete participation, country performance, gender representation, and medal distribution.

The goal of this project was to transform raw Olympics data into clear, interactive, and actionable insights.

## Key Analysis

* Gender bias and gender participation analysis
* Best-performing countries
* Top-performing athletes
* Participant growth by year
* Historical trend analysis
* Medal distribution analysis
* Country-wise Olympic performance
* Year-wise Olympic participation trends

## Dashboard Highlights

The dashboard allows users to explore Olympic data across different years and analyze how participation, medal performance, and gender representation have evolved over time.

## Tools Used

* Power BI
* Power Query
* DAX
* Data Cleaning
* Data Visualization
* Exploratory Data Analysis

## Key Learnings

Through this project, I strengthened my understanding of data transformation, trend analysis, dashboard design, DAX, and converting raw data into meaningful visual insights.
